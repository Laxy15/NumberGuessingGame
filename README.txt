Build and Unit Testing
----------------------
Use the Maven command after extracting the file to a specified folder.
mvn clean install
Note : For JUnit test, the application uses default number range from 1 to 100.

Setup
-----
Modify the config.properties file under ..\cfg to set the lower and higher number for the game
Note : The Application uses default number range from 1 to 20.

Run
---
Rename play.txt to play.bat
Then type "play" and press enter key
